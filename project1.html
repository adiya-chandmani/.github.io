VEX VR Robotics

In my VEX VR project, I used block coding to program a robot to solve mazes and sort colored disks. I learned to use loops and if-statements to make it work smartly.

VEX EXP Robotic

In my VEX EXP project, I built and programmed a robot to pick up and move balls. I used VEXcode to control its movement, arm, and sensors. I learned how to combine hardware and coding to solve tasks.

Introduction to Python

I learned Python basics like variables, loops, and functions. I made small projects like a guessing game and calculator, which helped me understand how coding and real programs work.

OnShape CAD

I used OnShape CAD to design 3D parts like gears and brackets. I learned to sketch, extrude, and add constraints. I built a working gear system and learned the design process from planning to improving.

micro:bit

I used micro:bit to make a simple game with LEDs, buttons, and motion sensors. One project was a rock-paper-scissors game that reacted when I shook it. I learned how to use sensors to create interactive devices.
