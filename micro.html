My micro:bit Projects

With micro:bit, I created a simple game using the LED display and buttons. I used sensors like the accelerometer to make the micro:bit respond to motion, and added sound output for effects. One of my projects was a rock-paper-scissors game that generated a random choice each time I shook the device. This helped me learn how sensors and inputs can be used to build interactive devices.
